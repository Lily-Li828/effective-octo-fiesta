earthquake, tsunami, and volcano eruptions.

About Our Data Visualizations
Histogram
Histogram is a graphical display of data using bars of different heights. It is similar to a Bar Chart, but a histogram groups numbers into ranges . The height of each bar shows how many fall into each range.

On this website, we use histogram to compare in between three disaster datas of a chosen country.
Line-chart
A line chart or line plot or line graph or curve chart is a type of chart which displays information as a series of data points called 'markers' connected by straight line segments.

On this website, we use a linechart to compare data of one disaster between one to three chosen countries.
